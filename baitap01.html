Giải thích logic tư duy

  • Quản lý index hiện tại, mỗi lần chuyển slide chỉ đổi giá trị index và translate track theo -index * 100%.
  • Để tránh ngoài giới hạn, dùng logic vòng tròn: index < 0 thì về slide cuối, index >= tổng slide thì về 0.
  • Auto slide dùng setInterval(3000); khi người dùng bấm next/prev hoặc dot thì reset timer để trải nghiệm mượt.
  • Tối ưu performance: chỉ animate bằng transform thay vì đổi left/margin, giúp render ổn định hơn.
  • Responsive: slide dùng aspect-ratio + width 100%, trên mobile vẫn giữ bố cục đẹp.
